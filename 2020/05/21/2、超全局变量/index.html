超级全局变量

PHP超级全局变量列表：
  • $GLOBALS
  • $_SERVER
  • $_GET
  • $_POST
  • $_FILES
  • $_ENV
  • $_COOKIE
  • $_SESSION

$GLOBALS — 引用全局作用域中可用的全部变量

$_SERVER — 服务器和执行环境信息

$_GET — HTTP GET 变量

$_POST — HTTP POST 变量

$_COOKIE — HTTP Cookies

$_FILE — HTTP 文件上传变量

$_REQUEST — HTTP Request 变量，用于收集HTML表单提交的数据

$_SESSION — Session 变量

$_ENV — 环境变量
